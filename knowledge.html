知识卡片

重点内容速记：字音字形、句式与病句。

字音：多音字判读

  • 根据语境判断语义确定读音。
  • 意义或用法不同导致音异，如“处(chu3/chu4)”。
  • 口语与书面语不同读音，如“血(xue4/xie3)”。
  • 人名地名或特定称谓读音特殊，如“单(shan4)”。

字音：常见易读错字（示例）

  • 鏖战 áo，白皑皑 ái，狭隘 ài
  • 瘠薄 bó，剥削 bō，薄纸 báo
  • 荸荠 bí，鄙视 bǐ，刚愎自用 bì
  • 纵横捭阖 bǎi，鞭笞 chī，整饬 chì

字形：常见错别字（示例）

  • 抱负(报) / 松弛(驰) / 醇香(淳)
  • 诨名(浑) / 装帧(桢) / 装潢(璜)
  • 渲染(喧) / 惯例(贯) / 诡异(鬼)
  • 赘述(缀) / 佝偻(瘘) / 牌匾(扁)

句式变换常考类型

  • 长句与短句
  • 肯定句与否定句
  • 主动句、被动句与“把”字句
  • 口语句式与书面语句式

仿用句式注意九点

话题、句式、修辞、结构、手法、色彩、逻辑、表达、字数。

选用句式答题步骤

  1. 读懂上下文，把握要点。
  2. 比较选项差异，结合语境寻找答案。
  3. 代入原文检验效果。

病句类型：表意不明

  • 词语多义导致歧义。
  • 停顿不同引发意义差别。
  • 指代不明导致歧义。

常见病句类型速记

  • 成分残缺 / 搭配不当
  • 语序不当 / 结构混乱
  • 重复赘余 / 前后矛盾
  • 表意不明 / 不合逻辑
