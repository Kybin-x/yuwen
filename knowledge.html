知识卡片

重点速记：字音字形、句式与病句、应用文写作。

字音：多音字判读

  • 根据语境判断语义确定读音。
  • 意义或用法不同导致音异。
  • 口语与书面语不同读音。
  • 人名地名或称谓读音特殊。

字形：常见错别字

  • 抱负(报) / 松弛(驰) / 醇香(淳)
  • 诨名(浑) / 装帧(桢) / 装潢(璜)
  • 渲染(喧) / 惯例(贯) / 诡异(鬼)
  • 赘述(缀) / 佝偻(瘘) / 牌匾(扁)

句式变换常考类型

  • 长句与短句
  • 肯定句与否定句
  • 主动句、被动句与“把”字句
  • 口语句式与书面语句式

病句类型速记

  • 成分残缺 / 搭配不当
  • 语序不当 / 结构混乱
  • 重复赘余 / 前后矛盾
  • 表意不明 / 不合逻辑

应用文写作卡片

按文档 17 类应用文整理，每类要点一句话。

便条

格式：称呼、正文、署名、日期；内容清楚、礼貌。

单据

格式：名称、正文、署名、签发日期；金额用大写。

启事

格式：标题、正文、落款；信息具体准确。

通知

格式：标题、主送机关、正文、落款；事项分条列款。

计划

格式：标题、正文、落款；强调目标、步骤与措施。

请示

格式：标题、主送机关、正文、落款；一事一文。

海报

格式：标题、正文、落款；语言简洁、鼓动性强。

简报

格式：报头、报体、报尾；内容真实客观。

会议纪要

格式：标题、文号、导言、正文、结尾；反映一致意见。

会议记录

格式：标题、组织情况、正文、落款；真实完整。

总结

格式：标题、正文、落款；成绩经验+不足展望。

调查报告

格式：标题、正文、署名；用材料说明观点。

说明书

格式：标题、正文、附文；科学实用、语言简明。

广告词

格式：标题、正文、随文；真实合法、重点突出。

书信

格式：称呼、正文、结尾、落款；称呼后用冒号。

求职应聘信

开头直陈求职意向，正文突出能力与经历。

合同

必备条款：标的、数量、价款等核心信息。
